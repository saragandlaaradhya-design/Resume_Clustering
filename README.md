# AI Resume Clustering System

## 📌 Overview

This project is a web-based application that uses Machine Learning (K-Means Clustering) to group resumes based on extracted skills.

## 🚀 Features

* Resume upload (multiple files)
* Skill extraction using NLP
* Resume clustering using K-Means
* Resume scoring system
* Job description matching
* Top candidate ranking
* Graph visualization

## 🛠 Technologies Used

* Python
* Flask
* Scikit-learn
* HTML, CSS, JavaScript

## ▶️ How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000

## 📊 Output

* Clustered resumes
* Candidate ranking
* Visualization graph

## 📁 Sample Files

Sample resumes are included in the uploads folder for testing.
